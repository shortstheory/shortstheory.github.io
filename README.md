# Arnav's Weblog

This site now uses the [Chirpy Jekyll theme](https://github.com/cotes2020/jekyll-theme-chirpy).

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Use Ruby `3.0+` locally. The GitHub Pages workflow builds the site with Ruby `3.4`.

## Deployment

GitHub Pages deployment is configured through [`.github/workflows/pages-deploy.yml`](/Users/arnavdhamija/projects/shortstheory.github.io/.github/workflows/pages-deploy.yml).
